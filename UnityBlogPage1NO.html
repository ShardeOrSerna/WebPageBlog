Unity

Kodet av: Andreas Phillip Krogrud.

Skrevet av: Edgaras Savicius.

Unity 5 er en engine som man bruker til å lage spill. Unity 5 kommer med monodevelop som lar deg skrive scripts med languages som c#,boo og javascript.boo er Unity sit eget språk men ingen bruker det. C# er den som blir brukt mest. mens javascript blir brukt det er ikke så populert som C#.

I Unity så kan du lage spill til mobil,pc,web og mange andre platformer i 3D og 2D.

https://unity3d.com/5

Unity er gratis,men hvis du vil så kan du kjøpe den pro versjonen som hjelper deg med mange ting.

Unity 5

Det finnes ganske mange tutorials for unity 5 de meste er pa c# men du kan finne noen som er javascript, men boo ingen trenger å lære den så du kan glemme den.

Jeg og Andreas har lagd et spill med unity 5(du kan finne mange prosjekter med navn test etc.) Jeg har også laget spill på mobillen så vis du vil gjøre det så kan du. Vel det er ikke noe mer med unity 5.Så det jeg vill si er at du bør bruke unity 5 for å lage spill.
